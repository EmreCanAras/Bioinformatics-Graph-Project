# Bioinformatics-Graph-Project
In this project, we developed a Java program to represent the STRING Protein-Protein Interaction (PPI) Network with a Graph ADT and provide some specified functionalities.

# Graphs in Bioinformatics (Java) 

This project focuses on modeling and analyzing biological data using **Graph Abstract Data Types (ADT)**. Developed as part of the Computer Engineering curriculum at Dokuz Eylül University, it demonstrates how complex biological relationships can be solved using efficient data structures.

##  Overview
In bioinformatics, genomic sequences or protein interactions often form intricate networks. This project:
* Represents biological data using **Graph** data structures.
* Optimizes memory management using [e.g., Adjacency List / Adjacency Matrix] representations.
* Follows Object-Oriented Programming (OOP) principles in Java.

##  Technical Specifications
- **Language:** Java
- **Data Structures:** Graph ADT, [e.g., Queue, Stack, Hash Map]
- **Algorithms:** BFS, DFS, Dijkstra’s Algorithm
- **Environment:** IntelliJ IDEA
